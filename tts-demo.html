🚀 Setup Instructions

Follow these steps to get the Gradio TTS app running:

1

Install Python Dependencies

Install all required Python packages from the requirements.txt file:

pip install -r requirements.txt

This installs Gradio, gTTS, pydub, scipy, and other dependencies.

2

Start the Gradio Server

Run the Gradio app from your project directory:

python tts_gradio_app.py

The app will start on http://localhost:7860 with a public share URL.

3

Embed in Your Website

Copy the public Gradio URL and embed it as an iframe, or use the API directly. The Gradio interface will appear below once the server is running.

4

Optional: Advanced TTS APIs

To use paid APIs like ElevenLabs or OpenAI:

ElevenLabs: pip install elevenlabs

OpenAI: pip install openai

Azure: pip install azure-cognitiveservices-speech

Then edit the generate_speech() function in tts_gradio_app.py to use your API key.

🔌 Integrating Premium APIs

ElevenLabs Integration

ElevenLabs provides high-quality voices with emotion control:

  1. Sign up at https://elevenlabs.io
  2. Get your API key from dashboard
  3. In tts_gradio_app.py, add:
# Example ElevenLabs integration from elevenlabs import client client.api_key = "YOUR_API_KEY_HERE"

response = client.generate(
  text=text,
  voice="Adam",
  model="eleven_monolingual_v1"
)

OpenAI Text-to-Speech

OpenAI offers neural TTS with multiple voices:

  1. Get OpenAI API key from platform.openai.com
  2. Usage example:
from openai import OpenAI
client = OpenAI(api_key="YOUR_API_KEY")

response = client.audio.speech.create(
  model="tts-1-hd",
  voice="alloy",
  input=text
)
response.stream_to_file("output.mp3")

🔧 Troubleshooting

Common Issues

  • Port 7860 already in use: Run on different port: change server_port in tts_gradio_app.py
  • FFmpeg not found: Install ffmpeg from system package manager
  • gTTS not generating audio: Check internet connection. gTTS requires online access.
  • Gradio iframe not loading: Ensure Gradio server is running on the correct address
